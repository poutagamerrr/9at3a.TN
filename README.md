## 9at3a.TN Backend (Node/Express/MongoDB)

### Setup
- **Install dependencies**: `npm install`
- **Environment**: copy `.env.example` to `.env` and adjust values (MongoDB URI, JWT secret, admin codes, port).

### Development
- **Run dev server**: `npm run dev` (Nodemon auto-restart)
- **Run in production**: `npm start`

The API exposes authentication, product, cart, order, and user management endpoints for the 9at3a.TN e-commerce platform.
